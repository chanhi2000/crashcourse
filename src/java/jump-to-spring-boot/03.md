::: info 이 장의 목표

- SBB를 상용 게시판 수준으로 개발한다.
- 부트스트랩을 적용하여 서비스를 더 보기좋게 만든다.
- 게시물 등록, 삭제, 수정부터 로그인, 로그아웃, 페이징, 검색까지 게시판을 완벽하게 만든다.
